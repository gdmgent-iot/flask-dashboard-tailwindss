# Flask Dashboard

# Systeemvereisten

- node js & npm

## Indien node geïnstalleerd

Controleren of node en npm geïnstalleerd zijn:

```bash
node -v
npm -v
```

Indien niet geïnstalleerd, voer volgende commando's uit in de terminal:

```bash
sudo apt install nodejs
sudo apt install npm
```
